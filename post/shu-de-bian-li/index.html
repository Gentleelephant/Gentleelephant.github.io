    N叉树的前序遍历（迭代）

    class Solution {
    public List<Integer> preorder(Node root) {

        Stack<Node> stack = new Stack<>();
        List<Integer> res = new ArrayList<>();
        if(root == null){
            return res;
        }
        
        stack.push(root);
        while(!stack.isEmpty()){
            Node temp = stack.pop();
            res.add(temp.val);
            // 把子节点逆序入栈
            Collections.reverse(temp.children);
            for(Node node:temp.children){
                stack.push(node);
            }
        }

        return res;

    }
}

    N叉树的前序遍历（dfs）

    class Node {
    public int val;
    public List<Node> children;

    public Node() {}

    public Node(int _val) {
        val = _val;
    }

    public Node(int _val, List<Node> _children) {
        val = _val;
        children = _children;
    }
}

class Solution {
    List<Integer> res = new ArrayList<>();
    public List<Integer> postorder(Node root) {
        dfs(root);
        return res;

    }
    public void dfs(Node root){
        if(root == null){
            return;
        }
        for(Node n:root.children){
            dfs(n);
        }
        res.add(root.val);
    }
}

class Solution {
    List<Integer> res = new ArrayList<>();
    public List<Integer> postorder(Node root) {
        if(root == null){
            return res;
        }
        method(root);
        Collections.reverse(res);
        return res;

    }
    // 迭代
    public void method(Node root){
        Stack<Node> stack = new Stack<>();
        stack.push(root);
        while(!stack.isEmpty()){
            Node temp = stack.pop();
            res.add(temp.val);
            for(Node n:temp.children){
                stack.push(n);
            }
        }
    }
}

    二叉树的前序遍历（迭代）

     public class TreeNode {
      int val;
      TreeNode left;
      TreeNode right;
      TreeNode(int x) { val = x; }
 }
class Solution {
    List<Integer> res = new ArrayList<>();
    public List<Integer> preorderTraversal(TreeNode root) {
        Stack<TreeNode> stack = new Stack<>();
        if(root == null){
            return res;
        }
        stack.push(root);
        while(!stack.isEmpty()){
            TreeNode node = stack.pop();
            res.add(node.val);
            if(node.right != null){
                stack.push(node.right);
            }
            if(node.left != null){
                stack.push(node.left);
            }
        }
        return res;
    }
}

    二叉树的中序遍历（迭代）

    class Solution {
    public List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> list = new ArrayList<>();
        Stack<TreeNode> stack = new Stack<>();
        if(root == null){
            return list;
        }
        TreeNode node = root;
        while(!stack.isEmpty() || node != null){

            // 中序遍历左中右
            // 先把左子树节入栈
            while(node != null){
                stack.push(node);
                node = node.left;
            }

            TreeNode temp = stack.pop();
            // 中
            list.add(temp.val);
            // 右
            node = temp.right;

        }
        return list;
    }
}

    二叉树的后序遍历迭代

    public List<Integer> postorderTraversal(TreeNode root) {
        LinkedList<Integer> result = new LinkedList<>();
        Stack<TreeNode> stack = new Stack<>();
        while (root != null || !stack.isEmpty()) {
            if (root != null) {
                stack.push(root);
                result.addFirst(root.val);
                root = root.right;
            } else {
                root = stack.pop();
                root = root.left;
            }
        }
        return result;
    }
