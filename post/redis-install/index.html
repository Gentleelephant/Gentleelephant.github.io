    一、编译安装(前提要有gcc/g++环境)

    1. 进入官网复制链接：http://download.redis.io/releases/redis-5.0.8.tar.gz

    2. 在Linux下使用wget http://download.redis.io/releases/redis-5.0.8.tar.gz 命令下载Redis

    3. 然后使用tar -zxvf redis-5.0.5.tar.gz命令解压

    4. 执行make

    5. 执行meke install

    6. 安装完成后启动redis：redis-server redis-conf
