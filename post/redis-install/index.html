Redis安装

2020-03-15 2 min read # Redis

一、编译安装(前提要有gcc/g++环境)

  1. 进入官网复制链接：http://download.redis.io/releases/redis-5.0.8.tar.gz

  2. 在Linux下使用wget http://download.redis.io/releases/redis-5.0.8.tar.gz 命令下载Redis

  3. 然后使用tar -zxvf redis-5.0.5.tar.gz命令解压

  4. 执行make

  5. 执行meke install

  6. 安装完成后启动redis：redis-server redis-conf

二、通过docker安装(centos)

docker安装好之后，先启动docker，然后直接运行命令：

docker run --name codezp-redis -d -p 6379:6379 redis --requirepass root

docker上的redis启动成功后，在宿主机器上有redis-cli的情况下可以直接运行命令连接：

redis-cli -a 123

如果宿主机器没有redis-cli也可以运行命令：

docker exec -it codezp-redis redis-cli -a root

三、通过命令直接安装(Ubuntu)

  1. apt-get update(root用户，不是root用户需要先su，或者sudo)

  2. apt-get install redis-server

  3. 启动redisredis-server

  4. 客户端连接redis-cli

  5. 出现：127.0.0.1:6379> 就成功了

  6. 可以在 /etc/redis目录下修改redis.conf文件

      • 二、通过docker安装(centos)
      • 三、通过命令直接安装(Ubuntu)
